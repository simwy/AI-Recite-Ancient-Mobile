# 默写纸预览与打印排版重构设计

## 目标

去掉所有 label 前缀，简化默写纸的预览卡片和 PDF 打印排版，使其更接近正式试卷样式。

## 新布局结构

```
         静夜思              ← 标题居中，直接显示古文标题
        唐 · 李白            ← 副标题居中，显示朝代·作者

初级默写              abc123  ← 左：默写级别值，右：文章ID值
─────────────────────────────  ← 分隔线
床前＿＿光，疑是＿上霜。       ← 正文内容
举＿望＿＿，低＿思故＿。
```

## 变更点

### 去掉的内容
- "古文默写纸" 大标题（PDF）
- 所有 label 前缀：`标题：`、`作者：`、`文章ID：`、`正文：`、`默写级别：`

### 新增/调整
- 标题直接居中显示
- 副标题（朝代·作者）居中显示
- 级别 + 文章ID 同一行，左右分布，位于分隔线之上
- 分隔线之后直接是正文

## 修改范围

1. `pages/ancient/dictation.vue` — paper-card 模板和样式
2. `uniCloud-alipay/cloudfunctions/gw_dictation-print-pdf/index.js` — PDF 头部生成逻辑
